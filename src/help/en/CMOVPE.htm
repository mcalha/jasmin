Command: CMOVPE - Move if parity even (PF=1)
Arguments: DEST: general-purpose register (16 or 32 bit); SRC: general purpose register (16 or 32 bit) or memory location
Usage: CMOVPE DEST,SRC
Effects: If (PF=1): DEST := SRC; Copies the source value (second argument) to the destination (first argument) if parityflag is set. Otherwise the programm continues with the next operation.
Flags to be set: none
