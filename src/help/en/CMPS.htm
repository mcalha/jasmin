Command: CMPS - Compare String Operands
Arguments: DEST, SRC: Memory Locations to be compared
Usage: CMPS DEST, SRC
Effects: Compares byte/word/doubleword at address DS:(E)SI with byte/word/doubleword at address ES:(E)DI and sets the status flags accordingly
Flags to be set: CF, OF, SF, ZF, AF, and PF
