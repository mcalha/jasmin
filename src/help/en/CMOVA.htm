Command: CMOVA - Move if above (CF=0 and ZF=0)
Arguments: DEST: general-purpose register (16 or 32 bit); SRC: general purpose register (16 or 32 bit) or memory location
Usage: CMOVA DEST,SRC
Effects: If (CF=0 and ZF=0): DEST := SRC; Copies the source value (second argument) to the destination (first argument) if carryflag and zeroflag are not set. Otherwise the programm continues with the next operation.
Flags to be set: none
