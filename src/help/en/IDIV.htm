Command: IDIV - Signed integer division
Arguments: SRC: register or memory location
Usage: IDIV SRC
Effects: EAX:=EDX:EAX / SRC; EDX:=rest of division Signed division of registers (see below) by src
Flags to be set: none
Misc: If SRC is 8 bit AX is divided, result is stored in AL and rest is stored in AH If SRC is 16 bit DX:AX is divided, result is stored in AX and rest is stored in DX If SRC is 32 bit EDX:EAX is divided, result is stored in EAX and rest is stored in EDX
