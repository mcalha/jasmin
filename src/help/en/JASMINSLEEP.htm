Command: JASMINSLEEP - Let Jasmin pause for a given number of milliseconds
Arguments: An immediate, or a register, or a memory location specifying how long to pause, in milliseconds.
Usage: JASMINSLEEP number_of_msecs
Effects: WARNING: THIS COMMAND IS NOT PART OF A REAL PROCESSOR'S INSTRUCTION SET! Please be aware that any program using it will only run inside Jasmin.
Flags to be set: none
